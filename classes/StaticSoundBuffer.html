Class StaticSoundBufferAbstract

Abstract class representing a static sound buffer.

A static sound buffer is a fully downloaded and decoded array of audio data that is ready to be played.

Static sound buffers can be reused multiple times by different StaticSound instances.

Static sound buffers are created by the CreateSoundBufferAsync function.
